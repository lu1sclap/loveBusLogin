# 🚌 戀愛巴士來賓查詢系統

一個輕量、美觀且響應式的網頁系統。專為「戀愛巴士」活動設計，來賓只需在手機或電腦網頁上輸入自己的「綽號」，系統便會直接連線至 Google 試算表（Google Sheets）當作資料庫，即時撈取並顯示該名來賓的專屬填寫資料。

支援手機、平板與電腦瀏覽（Responsive Web Design）。

## 🛠️ 功能特點

- **無後端架構**：完全使用純前端（HTML/CSS/JavaScript）技術，免去租用伺服器的成本。
- **試算表即資料庫**：直接讀取 Google 表單連動的試算表，後台管理與修改資料極其方便。
- **行動裝置優化**：針對手機瀏覽進行 UI 設計，完美適應各種螢幕尺寸。
- **獨立隱私過濾**：比對成功後，自動隱藏表單預設的「時間戳記」等無關欄位，只呈現有溫度的活動來賓資訊。
